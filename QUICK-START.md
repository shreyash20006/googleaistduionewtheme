# TGPCOP NOTES - Quick Start Guide

## 🚀 5-Minute Setup

This theme **automatically connects** to your existing Shopify store. Follow these simple steps:

### Step 1: Upload Theme (2 minutes)

1. **Compress this folder** to ZIP
2. Go to: **Shopify Admin** → **Online Store** → **Themes**
3. Click **"Upload theme"**
4. Select ZIP file
5. Click **"Publish"**

✅ **Done!** Your products and collections are now displaying automatically.

### Step 2: Add Logo (1 minute)

1. Click **"Customize"** on your theme
2. Click **"Header"** section
3. Upload your logo image
4. Save

### Step 3: Configure Homepage (2 minutes)

1. In Theme Editor, go to **Homepage**
2. Add sections:
   - **Dynamic Featured Collections**
   - **Dynamic Bestsellers**
3. For each section:
   - Click the section
   - Select which collection to display (or leave empty for auto-display)
   - Customize heading text
4. Save

## ✅ That's It!

Your theme is now:
- ✅ Displaying your existing products
- ✅ Showing your collections
- ✅ Using your product images
- ✅ Displaying correct prices
- ✅ Working with your cart
- ✅ Connected to your checkout

## 🎯 Optional Enhancements

### Add Product Tags for Badges

Go to any product and add tags:
- `bestseller` → Shows 🔥 Bestseller badge
- `new` → Shows ✨ New badge
- `instant` → Shows ⚡ Instant PDF badge

### Create Recommended Collections

If you haven't already, create these collections:
- Semester 1-8
- Bestsellers
- Handwritten Notes
- PYQs
- MCQs
- Combo Packs

The theme will automatically display them!

### Set Compare-at Prices

For any product:
1. Edit product
2. Set "Compare at price" (original price)
3. Set "Price" (sale price)
4. Save

The theme will automatically show the discount percentage!

**Products not showing?**
- Ensure products are published
- Check they're in a collection

**Collections empty?**
- Add products to collections
- Verify collection is not empty

**Cart not working?**
- Clear browser cache
- Try different browser
